                        INTRODUCTION

Rationale of the Study

  Programming has become one of the most essential skills in today's digital era, as it

fosters problem-solving, logical thinking, and creativity. However, many beginners--especially

learners who are interested in entering computing-related fields--find programming concepts

intimidating and difficult to grasp due to their abstract nature and the complexity of syntax.

Traditional teaching methods often emphasize theory, lectures, and memorization over active,

hands-on practice, which can result in low motivation, reduced engagement, and poor learning

outcomes (Robins et al., 2003). Due to this, many students struggle in introductory programming

courses, resulting in high failure rates and difficulty in mastering foundational concepts.

To address these challenges, game-based learning has emerged as an effective strategy to enhance
engagement, motivation, and conceptual understanding. Educational games create an interactive
environment where learners can explore concepts at their own pace while receiving immediate
feedback--an element that has been proven to support deeper learning (Plass et al., 2015). By
transforming abstract programming fundamentals into puzzles and challenges, students can learn
by doing rather than passively listening, which promotes active learning and improves retention
of knowledge (Prensky, 2001; Kapp, 2012).

Despite the benefits of existing game-based learning tools, many of them focus heavily on syntax
or complex programming tasks, making them overwhelming for complete beginners.
Additionally, most tools are not designed specifically to address the learning difficulties
experienced by junior high, senior high, or early college students who are still developing
foundational computational thinking skills. There is a need for an educational game that
introduces programming logic in a simplified, visual, and enjoyable way--one that helps reduce
anxiety, increases motivation, and builds confidence before students transition to writing actual
code.

The proposed educational puzzle game, Code Cat, aims to bridge this gap by offering a playful
and accessible platform for learning programming fundamentals. By guiding a character through
challenges using logic-based actions, learners can develop essential problem-solving and
sequencing skills in an environment that feels enjoyable rather than intimidating. This approach
                                                                                                                     2

supports students who may struggle with traditional instruction and provides an alternative
method that aligns with modern, learner-centered educational practices. Furthermore, by
deploying Code Cat as a web-based platform, the study ensures that these educational tools are
readily accessible to any student with a computer and internet access, removing technical barriers
to entry.

Code Cat is designed to bridge the gap between fun and learning. The game aims to introduce
programming logic and problem-solving skills in a simple, engaging, and accessible way.
Through interactive puzzles and game-based scenarios, learners can develop a deeper
understanding of programming basics while enjoying the process. This approach not only
promotes active learning but also reduces the fear of coding by presenting it in a playful and less
intimidating format.

Ultimately, this study is grounded on the belief that integrating educational games into learning
can significantly improve the way programming is taught and learned. By focusing on
engagement, interactivity, and practical application, Code Cat seeks to support beginners in
building a strong foundation in programming fundamentals while fostering a positive attitude
toward coding.

Objectives of the Study

General Objective

The general objective of this study is to analyze, design, and develop Code Cat, which will serve
         as an engaging tool to help avid learners understand basic programming concepts through
         interactive and game-based learning.

Specific Objectives:

    1. Identify the current or existing learning approach in programming education in terms of:
           1.1. The teaching and learning process used for introducing basic programming

         concepts; and
           1.2. The problems encountered by beginners, such as difficulty in understanding

         sequencing, loops, and conditionals, and low learner engagement.
    2. Determine the mechanism or technique to be utilized for the proposed system.

          2.1. Puzzle-based learning as a method for teaching programming fundamentals;
                                                                                                                     3

          2.2. Interactive gameplay to support hands-on practice and exploration; and
          2.3. Immediate feedback mechanisms to reinforce learning and improve comprehension.
    3. Define the best features of the proposed system, including an intuitive user interface,
         progressive difficulty levels, interactive puzzles, and game-based feedback that support
         beginners with little or no background in programming.
    4. Establish the level of acceptability of the proposed system in terms of usability,
         engagement, and effectiveness as a supplementary instructional tool for teaching
         programming fundamentals.

Scope and Limitations of the Study

         This study focuses on the design and development of Code Cat. The game will primarily
cover basic programming concepts, including sequencing, loops, and conditionals, which are
essential for building a foundation in coding. It is intended for beginners, particularly students
with little or no prior programming experience. The study emphasizes game-based learning,
where players solve interactive puzzles that represent real-world programming logic in a
simplified and engaging manner.

The target users of this study are junior high school and senior high school students, although it
can also apply to anyone interested in learning programming fundamentals in an accessible
format. The study also includes the design, development, and initial evaluation of the game to
assess its usability and effectiveness as an educational tool.

Scope of the Study

         Game Coverage ­ The game will focus on introducing fundamental programming
concepts such as sequencing, loops, variables, arrays, functions, and conditionals. Advanced
topics like algorithms, data structures, and object-oriented programming will not be included.

     Target Users ­ The intended users of the game are eager learners who are interested in
         learning programming, particularly junior high school and senior high school students,
         though it may also be useful for other novice self-learners.

     Learning Approach ­ The study emphasizes the use of game-based learning strategies,
         where programming concepts are presented in puzzle form to make learning interactive,
         engaging, and less intimidating.
                                                                                                                     4

     Platform ­ To maximize accessibility in school computer laboratories, the game will be
         developed as a Web-First Application. It will be accessible via standard desktop web
         browsers (e.g., Google Chrome, Microsoft Edge) without the need for installation. This
         approach ensures the game can be used on any operating system (Windows, macOS,
         Linux) and bypasses the administrative restrictions often found in university computer
         labs.

     Evaluation ­ The study will include initial testing and assessment of the game to
         determine its usability, educational effectiveness, and ability to motivate learners. The
         evaluation will be conducted within a limited group of participants.

     Rewards and Achievement System ­ The game will include a rewards system to reinforce
         motivation and encourage continued learning. Players will be able to earn achievements
         for completing levels, solving puzzles efficiently, or demonstrating mastery of concepts.
         These rewards are designed to foster a sense of accomplishment, support progression, and
         increase engagement throughout the learning experience.

     Development Scope ­ The project will cover the planning, design, and implementation of
         the game, focusing on user interface design, puzzle development, and basic functionality.
         Features such as multiplayer modes, online integration, or advanced graphics are beyond
         the scope of this study.

Limitations of the Study

         This study presents several limitations that may influence the generalizability and depth
of its findings. First, the game focuses only on fundamental programming concepts such as
sequencing, loops, and conditionals. As a result, learners seeking advanced programming topics
or more complex problem-solving activities may find the game insufficient for higher-level skill
development.

The evaluation of the game will be conducted with a limited group of participants, which may not
represent all potential users. The results will therefore focus on short-term usability and learning
effectiveness rather than long-term mastery of programming skills.

Additionally, the game will be developed for well-known platforms only for accessibility, and
may not be compatible with all other devices or operating systems. Other advanced features, such
as multiplayer functionality, extended levels, or online integration, are beyond the scope of this
study due to time and resource constraints.
                                                                                                                     5

Finally, this study will be conducted within the University of Cebu ­ Main Campus, focusing
primarily on students who are beginners in programming. The participants will be limited to those
enrolled at this institution to ensure ease of coordination, accessibility, and data collection.
Therefore, the findings of the study may primarily reflect the experiences and learning behaviors
of students at the University of Cebu ­ Main Campus and may not fully represent learners from
other campuses, universities, or regions.

Definition of Terms

         For clarity and better understanding of the study, the following terms are defined
operationally as they are used throughout the research:

         Code Cat - An educational, web-based puzzle game developed in this study to help
beginners learn basic programming concepts through interactive and game-based activities.

         Programming Fundamentals - The basic concepts of programming are introduced in
the game, including sequencing, loops, and conditionals, which form the foundation of logical
problem-solving in coding.

         Game-Based Learning - An instructional approach that integrates educational content
into gameplay elements such as puzzles, challenges, and rewards to promote active learning and
learner engagement.

         Puzzle-Based Learning - A learning strategy used in Code Cat where programming
concepts are presented as logical puzzles that require players to think critically and solve
problems step by step.
                                                                                                                     7

         Sequencing - A programming concept that refers to executing instructions in a specific
and logical order. In the game, this is represented by arranging actions correctly to complete a
task.

         Loops - A programming concept that allows a set of instructions to be repeated multiple
times. In Code Cat, loops are introduced through puzzles that require repeated actions to achieve
goals efficiently.

         Conditionals - A programming concept that involves decision-making using conditions
such as "if" and "else." In the game, conditionals are represented through challenges that require
players to choose actions based on given conditions.

         User Interface (UI) - The visual layout and interactive elements of the game that allow
players to navigate levels, view instructions, and interact with puzzles easily and intuitively.

         Beginner Learners - Individuals with little or no prior experience in programming,
particularly junior high school and senior high school students, who are the primary target users
of the game.

         Web-Based Application - A software application that runs on a web browser and does
not require installation. Code Cat is developed as a web-based platform to ensure accessibility
across different devices and operating systems.

         Learner Engagement - The level of interest, motivation, and active participation
demonstrated by learners while interacting with the game and completing programming puzzles.

         Immediate Feedback - Responses provided by the system after a player's action or
puzzle attempt, informing the learner whether the solution is correct and guiding improvement.

         Rewards and Achievement System - A feature of the game that provides incentives
such as badges or achievements to motivate learners, reinforce progress, and encourage continued
participation.

         Usability - The degree to which the game is easy to use, understand, and navigate by
beginner learners without requiring external assistance.

         Educational Effectiveness - The extent to which Code Cat helps learners understand and
apply basic programming concepts through gameplay.

                 REVIEW OF RELATED LITERATURE AND STUDIES

Review of Related Literature

         Game-based learning has emerged as an increasingly effective approach in modern
education, particularly in teaching technical skills such as programming. Prensky (2001) explains
that digital games can improve learner motivation, engagement, and knowledge retention by
integrating educational content with interactive play. Because programming is often viewed as
challenging for beginners, this approach helps reduce learning barriers by making abstract
concepts more accessible. Kapp (2012) further notes that game-based environments support
learning by offering immediate feedback and meaningful problem-solving scenarios, enabling
students to understand and apply programming logic.

Several studies highlight the effectiveness of educational games in teaching computational
thinking. Grover and Pea (2013) suggest that visual and interactive representations of code help
learners grasp abstract programming concepts more easily. Moreover, games that use block-based
or sequence-based systems simplify coding syntax and encourage experimentation without fear of
errors.

The integration of puzzles into educational games has also proven beneficial. Gee (2007) explains
that puzzles encourage critical thinking, strategic planning, and iterative learning -- key cognitive
skills required in programming. Therefore, puzzle-based coding games like Code Cat align well
with modern pedagogical methods that prioritize learning through discovery and practice rather
than memorization.

Local Literatures

         In the Philippines, the rise of digital learning tools has encouraged educators to use
game-based platforms in teaching technical subjects. According to Dela Cruz (2019), Filipino
students tend to respond positively to visual and interactive media, which promote better
comprehension and engagement in STEM-related lessons. Educational institutions have begun
integrating simple coding games, such as Code.org's Hour of Code, into basic computer education
curricula to introduce programming fundamentals at an early age.
                                                                                                                     9

Locally developed educational games, such as Math Run PH and Algeo PH, have shown that
gamified learning applications can make abstract subjects more approachable to Filipino learners.
Similarly, Code Cat aims to follow this trend by combining visual storytelling and gameplay
mechanics to teach programming logic in a fun and culturally relatable way.

Moreover, a study by Santos and Villanueva (2020) at the University of the Philippines found that
local students learn better when lessons are delivered through game-based activities that simulate
real-world logic and problem-solving. This supports the potential effectiveness of Code Cat as an
innovative educational tool for programming literacy.

Foreign Literature

         Internationally, game-based programming tools like Scratch, Lightbot, and CodeCombat
have demonstrated the power of interactive learning. Resnick et al. (2009) developed Scratch to
introduce programming through drag-and-drop coding blocks, allowing children to visualize code
structure without syntax complexity. This approach has inspired many educators to adopt similar
systems to teach programming logic to beginners.

Lightbot, created by Danny Yaroslavski, uses puzzle levels that require players to control a robot
using programming-like commands such as loops and conditionals. The game's design proves
how simple mechanics can effectively introduce core programming concepts. Likewise,
CodeCombat employs role-playing elements where players write real code to control their
characters, bridging the gap between fun and learning.

These foreign examples provide a strong foundation for Code Cat, which seeks to combine
puzzle-solving, character-driven gameplay, and visual programming to create an enjoyable and
educational experience for Filipino students.

Review of Related Studies

         A study by Fessakis, Gouli, and Mavroudi (2013) found that using digital games to teach
programming in primary education improves logical reasoning and engagement levels compared
to traditional lectures. The authors highlight that visual and interactive methods help reduce
learners' fear of programming.

In a local context, Mendoza (2021) conducted research on Gamified Learning Environments for
IT Education and reported that students exposed to gamified exercises performed better in coding
                                                                                                                   10
assessments and demonstrated greater interest in pursuing programming courses. Similarly, a
study by Javier and Ramos (2020) concluded that puzzle-based games enhance students'
problem-solving and algorithmic thinking skills.
These studies collectively support the concept behind Code Cat -- that gamified environments
can transform abstract programming principles into interactive challenges, improving both
learning outcomes and student motivation.
Self-Determination Theory

                                    Figure 1. Self-Determination Theory
Self-Determination Theory (SDT), developed by Deci and Ryan, explains human motivation by
emphasizing three fundamental psychological needs: autonomy, competence, and relatedness.
When these needs are satisfied, learners are more likely to exhibit intrinsic motivation, sustained
engagement, and improved learning outcomes. This theory serves as a guiding framework in the
design and development of Code Cat.
Autonomy is supported in Code Cat by allowing players to control the pace of their learning and
choose how to approach programming challenges. Players are free to experiment with different
                                                                                                                    11

solutions to puzzles, encouraging exploration and independent problem-solving rather than
passive instruction.

Competence is addressed through progressively structured levels that gradually introduce
programming concepts such as sequencing, loops, and conditionals. Immediate feedback is
provided after each puzzle attempt, allowing players to understand mistakes, refine their
solutions, and experience a sense of achievement as they master new concepts.

Relatedness is incorporated by presenting challenges within a narrative-driven and
character-based game environment. The use of a friendly game character and contextual scenarios
helps learners feel emotionally connected to the learning experience, making programming
concepts more approachable and engaging.

By integrating the principles of Self-Determination Theory, Code Cat promotes intrinsic
motivation and active learning. This theoretical foundation supports the game's objective of
making programming education enjoyable, meaningful, and effective for beginners.

                                DESIGN AND METHODOLOGY

Research Design Method

         This study utilized a developmental and descriptive research design to guide the creation
and evaluation of Code Cat, an educational puzzle game aimed at helping students learn
programming fundamentals in an interactive and engaging way. The developmental aspect
focused on the step-by-step process of designing and building the game, while the descriptive
aspect dealt with assessing its usability, functionality, and educational effectiveness among
students.

         The researchers followed a structured approach that included planning, designing,
developing, testing, and evaluating the system. During the planning stage, the objectives, target
users, and core features of the game were identified. The design stage involved creating the
interface layout, gameplay structure, and logic flow to align with programming learning goals.
The development stage focused on implementing the game features using appropriate
programming tools and testing them for functionality.

         Once the prototype is completed, it will undergo a structured evaluation process. For
learning effectiveness, the system will be tested by Information Technology and Computer
Science students enrolled in introductory programming courses. These participants will be
divided into two groups: an experimental group that will use the game as a supplemental learning
tool, and a control group that will continue with traditional instruction. Both groups will complete
pre-tests and post-tests to measure learning gains and compare the impact of the game-based
approach.

         To further validate usability and acceptability, User Acceptance Testing (UAT) will be

Network Design

         The network design for Code Cat follows a simple client­server architecture where
players access the game through their devices using a secure connection to the application server.
The application server handles game logic, user authentication, and communication with the
database server, which stores all game data such as user accounts, levels, puzzles, and progress.
Administrators connect to the same server through a secure interface to update content and
monitor player performance. To ensure reliability and security, the system uses HTTPS
encryption, firewalls, and controlled access to protect data and maintain smooth gameplay.

Network Model

         The network model for Code Cat follows a client­server structure designed to support
efficient gameplay and secure data management. In this model, players use their devices to access
the game through a secure connection to the application server, which processes game logic, user
authentication, and all interactions within the system. The application server communicates with a
dedicated database server that stores user information, levels, puzzles, code attempts, and
achievements. Administrators access the system through a secured portal on the same server,
allowing them to manage game content and monitor learner progress. To maintain reliability and
security, the network model incorporates HTTPS communication, firewalls, and restricted access
between servers. This setup ensures that the game runs smoothly for players while keeping all
data protected and centralized.

Network Topology

         The network topology for Code Cat utilizes a simplified client­server star topology to
ensure efficient communication and centralized control of game operations. In this setup, all
client devices--such as desktops, laptops--connect directly to the central application server,
which acts as the main hub of the system. The application server handles user requests, game
logic, and data processing while maintaining a secure connection to the database server, where all
game-related data is stored. Administrators also connect to the application server through a secure
interface for content management and monitoring. This star topology provides improved
reliability, as issues with one client do not affect others, and it allows for easier maintenance and
scalability. By centralizing communication through the main server, the system ensures smooth
gameplay, consistent performance, and secure data handling.
